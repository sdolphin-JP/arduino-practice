# Arduino - 圧電ブザーを鳴らしてみるお勉強

ドラクエ3の戦闘BGMの最初の方を流してみてたようだ。

# 1. HW
- 制御用装置は、「Arduino UNO R3」を使用する。
- 圧電ブザーは、そこらへんに転がっているものを使用する。

# 2. SW
- OSは、何でも良い。
- 開発環境は、「[Arduino IDE][def_url_aide]」を使用する。

# 3. 配線
- Digital#08 -- プラス
- GND -- マイナス
